# W3F — Phase 2 Proposal PDF Generator

> **WRO 2026 Racing Proposal** — Win First or Fail Fast

## Usage

1. Open `index.html` in a browser (Chrome recommended)
2. Preview individual pages: `[P1] [P2] [P3]` or `[ALL]`
3. Click **EXPORT PDF** to download the 3-page proposal

## Requirements

- Internet connection (for fonts, F1 hero image, CDN libraries)
- Modern browser (Chrome, Edge, Firefox)

- **Theme:** F1 Pit Wall Spec Sheet — luxury racing aesthetic
